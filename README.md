# QuantLab-Monte-Carlo-Risk-Engine
Institutional-style Monte Carlo Portfolio Risk Engine built in Python using live market data, correlated Geometric Brownian Motion, portfolio optimization, Value at Risk (VaR), Expected Shortfall (CVaR), and interactive visualizations.
# 📊 QuantLab – Monte Carlo Portfolio Risk Engine

> Institutional-style quantitative finance project for portfolio risk analysis using Monte Carlo Simulation and Modern Portfolio Theory.

---

# Project Overview

QuantLab is a Python-based portfolio risk engine that simulates future portfolio performance using **100,000 correlated Monte Carlo simulations**.

The application downloads live market data from Yahoo Finance, estimates covariance between assets, optimizes portfolios, and evaluates risk using institutional portfolio analytics.

---
